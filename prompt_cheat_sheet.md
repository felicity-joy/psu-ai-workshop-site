## สูตรเขียน Prompt 4 ข้อ

1. **บอกตรงๆ ว่าต้องการอะไร** — AI ไม่ใช่นักอ่านใจ
2. **ให้บริบท** — ตำแหน่งงาน วัตถุประสงค์ กลุ่มเป้าหมาย
3. **ยกตัวอย่างรูปแบบ** — ต้องการสรุปแบบไหน ยาวเท่าไร
4. **ปรับแก้ทีละขั้น** — หากผลไม่ตรง ให้บอกเพิ่ม

## ตัวอย่าง Prompt ตามประเภทงาน

### 1. สรุปรายงานการประชุม

```
คุณคือเลขานุการคณะเทคโนโลยีและสิ่งแวดล้อม ม.อ. ภูเก็ต
ช่วยสรุปบันทึกการประชุมต่อไปนี้เป็นรายงานการประชุม โดยมี:
1. หัวข้อการประชุม
2. วันเวลาและสถานที่
3. ผู้เข้าร่วมประชุม
4. ประเด็นสำคัญ 3-5 ข้อ
5. งานที่ต้องทำ พร้อมผู้รับผิดชอบและกำหนดเวลา

ข้อกำหนด: ภาษาทางการ กระชับ ไม่เกิน 400 คำ

[วางข้อความบันทึกการประชุม]
```

---

### 2. ร่างบันทึกข้อความ

```
คุณคือเจ้าหน้าที่งานเลขานุการของคณะเทคโนโลยีและสิ่งแวดล้อม ม.อ. ภูเก็ต
ช่วยร่างบันทึกข้อความขออนุมัติจัดกิจกรรม โดยมีรายละเอียด:
- ผู้ส่ง: ................
- ผู้รับ: ................
- เรื่อง: ................
- วัตถุประสงค์: ................
- กำหนดการ: ................
- งบประมาณ: ................

ข้อกำหนด:
- ภาษาราชการสุภาพ กระชับ
- มีโครงสร้าง: ที่ วันที่ เรื่อง เรียน เนื้อหา ขออนุมัติ ลงชื่อ
- ไม่เกิน 250 คำ
```

---

### 3. ร่างอีเมลแจ้งนักศึกษา/บุคลากร

```
คุณคือเจ้าหน้าที่ฝ่ายทะเบียนของคณะเทคโนโลยีและสิ่งแวดล้อม ม.อ. ภูเก็ต
ช่วยเขียนอีเมลแจ้ง [เรื่อง] โดยมีรายละเอียด:
- กลุ่มผู้รับ: ................
- กำหนดการ/วันสำคัญ: ................
- ขั้นตอนที่ต้องทำ: ................
- ช่องทางติดต่อหากมีข้อสงสัย: ................

ข้อกำหนด:
- ภาษาทางการแต่กระชับ
- มีหัวข้ออีเมลที่ชัดเจน
- ไม่เกิน 150 คำ
```

---

### 4. จัดระเบียบข้อมูลเป็นตาราง (ผ่านหน้าต่างแชท)

```
คุณคือเจ้าหน้าที่ฝ่ายวิเคราะห์ข้อมูล
ฉันมีข้อมูลที่บันทึกไม่เป็นระเบียบ ช่วยแปลงให้เป็นตารางที่อ่านง่าย ตามหลักการ:
- 1 คอลัมน์ = 1 ตัวแปร
- 1 แถว = 1 รายการ
- 1 ช่อง = 1 ค่า

ข้อมูลต้นฉบับ:
[วางข้อมูล]

ข้อกำหนด:
1. แยกคอลัมน์เป็น: ................
2. หากข้อมูลไม่ครบให้เว้นว่าง
3. แสดงผลในรูปแบบตารางที่อ่านง่าย
4. อธิบายสั้นๆ ว่าแก้ไขอะไรบ้าง
```

### 5. ร่างประกาศ/ข่าวสาร

```
คุณคือเจ้าหน้าที่ประชาสัมพันธ์ของคณะเทคโนโลยีและสิ่งแวดล้อม ม.อ. ภูเก็ต
ช่วยร่างประกาศ [เรื่อง] โดยมีรายละเอียด:
- กลุ่มเป้าหมาย: ................
- วันที่/เวลา/สถานที่: ................
- รายละเอียดสำคัญ: ................
- ช่องทางสมัคร/ติดต่อ: ................

ข้อกำหนด:
- ภาษากระชับ ชัดเจน น่าสนใจ
- ไม่เกิน 200 คำ
- เหมาะสำหรับโพสต์บนเว็บไซต์คณะ
```

### 6. ตอบอีเมล/ข้อซักถาม

```
คุณคือเจ้าหน้าที่ฝ่าย [ชื่อฝ่าย] ของคณะเทคโนโลยีและสิ่งแวดล้อม ม.อ. ภูเก็ต
ช่วยร่างการตอบอีเมลต่อคำถามต่อไปนี้:

คำถามของผู้ส่ง:
[วางคำถาม]

ข้อมูลที่ต้องการสื่อสาร:
- ................
- ................

ข้อกำหนด:
- โทนเป็นทางการแต่เป็นมิตร
- ให้ข้อมูลครบถ้วน ชัดเจน
- ความยาวไม่เกิน 200 คำ
```

## เทคนิคปรับปรุง Prompt

หากผลลัพธ์ไม่ตรงใจ ลองเพิ่ม:

- **ระบุบทบาท**: "คุณคือเจ้าหน้าที่ฝ่ายทะเบียนที่มีประสบการณ์ 10 ปี"
- **ระบุกลุ่มเป้าหมาย**: "ใช้ภาษาที่เข้าใจง่ายสำหรับนักศึกษาใหม่"
- **ระบุสิ่งที่ไม่ต้องการ**: "ไม่ต้องมีคำทักทายและลงท้ายแบบจดหมาย"
- **ให้ตัวอย่าง**: "เขียนในรูปแบบเดียวกับตัวอย่างนี้ ..."
- **ขอให้คิดเป็นขั้นตอน**: "วิเคราะห์ตามขั้นตอนต่อไปนี้: 1) ... 2) ... 3) ..."

## ข้อควรระวัง

- ไม่ใส่ข้อมูลส่วนบุคลากร รหัสผ่าน หรือข้อมูลลับของมหาวิทยาลัย
- ตรวจสอบผลลัพธ์ทุกครั้งก่อนนำไปใช้
- ใช้วิจารณญาณ — AI อาจให้ข้อมูลไม่ถูกต้อง
